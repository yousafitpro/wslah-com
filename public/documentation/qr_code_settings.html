QR Code Settings

How to modify QR code.

Click on QR Code showing like this.

  1. Modify QR code:

    Image >> Select image >> Change image size

    Color >> Select Only Single Color >> Select color & change color transparent

    Color >> Select other then Only Single Color>> Select first color & second color

    Color >> Select background color & change color transparent

    Style >> Select QR style

    Style >> Select eye style >> Select eye inner & outer color

    Size >> Change image size

Click on Save button to save QR details.

Click on Reset button to Revert your changes.

Click on Download button to Download you QR code in PNG format.
